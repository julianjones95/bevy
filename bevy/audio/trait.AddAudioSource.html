Expand description

A trait that allows adding a custom audio source to the object. This is implemented for [App][bevy_app::App] to allow registering custom Decodable types.

Required Methods

fn add_audio_source<T>(&mut self) -> &mut Selfwhere
    T: Decodable + Asset,

Registers an audio source. The type must implement Decodable, so that it can be converted to a [rodio::Source] type, and Asset, so that it can be registered as an asset. To use this method on [App][bevy_app::App], the audio and [asset][bevy_asset::AssetPlugin] plugins must be added first.
